# Notes-App
Command-line Notes utility to add, save, list and remove notes

## Running the app


### To get a note
```
 node app.js read --title="Hello"
 ```
 
 ### To list all notes
 ```
 node app.js list
 ```
 
 ### To add a note
```
 node app.js read --title="Hello" --body="World"
 ```
 
  ### To delete a note
```
 node app.js remove --title="Hello"
 ```
